# Knowledge Compounding

Knowledge compounding is a core principle of this wiki system: **the value of the knowledge base increases non-linearly as more sources are added.**

## How It Works

Each new source ingested:
1. Adds new pages or enriches existing ones
2. Creates new cross-references between existing concepts
3. Provides additional context for synthesizing answers to future queries
4. May resolve previously open questions or surface new ones

## Analogy

Like financial compounding, early investments (ingesting foundational sources) yield increasing returns over time as later sources build upon them. A wiki with 100 well-connected pages is more than 10x as useful as 10 isolated pages.

## Practical Implication

The more consistently sources are ingested, the faster and more powerfully the knowledge base grows. This rewards regular use of the system.
