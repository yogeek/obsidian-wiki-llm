# Obsidian Integration

The personal wiki is designed to be browsed and navigated using [Obsidian](https://obsidian.md), a Markdown-based knowledge management application.

## Why Obsidian

- Renders Markdown files natively
- Supports `[[wiki-links]]` for inter-page navigation
- Provides a **graph view** to visually explore connections between pages
- Works directly on the local filesystem — no sync required

## Setup

1. Open Obsidian
2. Select **Open folder as vault**
3. Point it to your `wiki/` directory
4. Wiki links and graph view will work automatically

## Tips

- Use the **Graph View** (`Ctrl/Cmd + G`) to explore entity relationships visually
- Use **Backlinks** panel to see what references a given page
- Tags defined in frontmatter are searchable within Obsidian
