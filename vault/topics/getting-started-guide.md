# Getting Started Guide

## Quick Start

Begin building your personal knowledge base with these steps:

### Step 1: Add Sources
Place your raw content in the `raw_sources/` directory:
- Documents you want to process
- Notes you've taken
- Articles you've saved
- Any text-based content

### Step 2: Run Ingestion
Execute the ingestion process to transform raw sources into wiki pages:
- The LLM analyzes your content
- Entities and topics are extracted
- Pages are generated with cross-references
- Knowledge structure emerges

### Step 3: Query the Wiki
Use the system to:
- Synthesize knowledge across sources
- Discover connections between concepts
- Generate insights from your information
- Answer questions using accumulated knowledge

### Step 4: Browse in Obsidian
Open your wiki directory in Obsidian to:
- Visualize the knowledge graph
- Navigate between connected pages
- Edit and enhance content
- Add personal annotations

## Tips for Success
- Start with a few high-quality sources
- Review generated pages for accuracy
- Add sources incrementally
- Use consistent naming conventions
- Regularly browse your growing wiki
