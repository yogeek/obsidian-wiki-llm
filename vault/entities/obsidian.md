# Obsidian

Obsidian is a local-first, Markdown-based knowledge management and note-taking application. It is the recommended interface for browsing this personal wiki.

## Key Features Relevant to This Wiki

- **Vault system** — Opens any folder of Markdown files as a knowledge base
- **Wiki links** — `[[page-name]]` syntax for inter-page navigation
- **Graph view** — Visual map of all pages and their connections
- **Frontmatter support** — Reads YAML frontmatter for metadata and tags
- **Backlinks** — Shows all pages that reference the current page
